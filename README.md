대학생을 위한 수업·과제 통합 관리 웹사이트입니다.
LMS, 카카오톡, 이메일에 흩어져 있는 수업 정보와 과제 마감일을 한 곳에 모아 관리할 수 있습니다.

## 이런 분께 필요해요

- 여러 과목 과제 마감일을 놓쳐서 곤란했던 적이 있다
- 강의자료 링크가 여기저기 흩어져서 찾기 번거롭다
- 오늘, 이번 주에 뭘 해야 하는지 한눈에 보고 싶다

## 무엇을 할 수 있나요

**오늘 할 일**
오늘 마감인 과제와 이번 주 마감인 과제를 한눈에 보여줍니다. 등록된 수업 수, 남은 과제 수도 함께 확인할 수 있습니다.

**수업 등록**
수업 이름, 교수님, 요일, 시간, 강의실, 색상을 입력해서 내 수업을 등록합니다. 색상을 지정해두면 과제와 자료를 볼 때 어느 수업 것인지 한눈에 구분할 수 있습니다.

**과제 관리**
수업을 선택하고 과제 제목, 유형(리포트/발표/퀴즈/기타), 마감 날짜와 시간을 입력합니다. 마감까지 며칠 남았는지(D-day)가 자동으로 표시되고, 완료한 과제는 체크 표시로 구분됩니다.

**강의자료**
수업별로 강의 PPT, 신택스, 공지 링크 등을 저장해두고 클릭 한 번으로 열어볼 수 있습니다.

## 사용 방법

1. **수업 등록** 탭에서 수강 중인 수업을 먼저 등록합니다.
2. **과제 관리** 탭에서 수업을 선택하고 과제와 마감일을 입력합니다.
3. **오늘 할 일** 탭에서 매일 마감 임박 과제를 확인합니다.
4. 다 한 과제는 동그란 체크 버튼을 눌러 완료 표시합니다.
5. 더 이상 필요 없는 항목은 ✕ 버튼으로 삭제합니다.

새 학기가 시작되면 화면 맨 아래 "전체 데이터 초기화" 버튼으로 깨끗하게 새로 시작할 수 있습니다.

## 알아두면 좋은 점

- 별도 회원가입이나 로그인이 필요 없습니다.
- 입력한 모든 데이터는 **현재 사용 중인 브라우저에만** 저장됩니다. 다른 기기나 다른 브라우저에서는 보이지 않으며, 브라우저 데이터를 지우면 함께 사라질 수 있습니다.
- 인터넷 연결 없이도 한 번 접속한 뒤에는 대부분 기능을 사용할 수 있습니다.
