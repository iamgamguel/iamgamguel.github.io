# 내 시간표 관리 (수업 등록 기능)

대학생을 위한 수업/과제 관리 웹앱의 1단계 — **수업 등록 기능** 구현입니다.

## 기능
- 수업 이름 / 교수명 / 강의실 등록
- 요일 다중 선택 (월~금)
- 수업 시간 입력 (시작~종료)
- 수업별 색상 태그 지정
- 등록된 수업 목록 조회 / 삭제
- 브라우저 localStorage에 자동 저장 (새로고침해도 유지)

## 실행 방법
`index.html` 파일을 더블클릭해서 브라우저로 열면 바로 사용 가능합니다.
별도 설치나 서버가 필요 없습니다.

## GitHub Pages로 배포하는 방법
1. GitHub에 새 저장소(repository)를 만듭니다.
2. 이 폴더의 `index.html`을 저장소에 업로드합니다.
3. 저장소 Settings → Pages → Source에서 `main` 브랜치, `/ (root)` 선택 후 저장합니다.
4. 몇 분 후 `https://[아이디].github.io/[저장소이름]/` 주소로 접속하면 확인할 수 있습니다.

## 사용 기술
- HTML / CSS / JavaScript (순수 바닐라, 프레임워크 없음)
- 데이터 저장: 브라우저 localStorage
